安装“导出财新网页”工具

请将下方的按钮拖拽到您的浏览器书签栏中：

导出财新网页

操作演示：

1. 确保 Chrome 浏览器书签栏已显示（快捷键 Ctrl+Shift+B / Cmd+Shift+B）

2. 用鼠标按住上方按钮，拖拽到书签栏松开即可。

安装油猴脚本

若浏览器未安装油猴插件，请根据浏览器类型，选择安装一个用户脚本管理器（Tampermonkey 或 Violentmonkey）

    Microsoft Edge：Tampermonkey 或 Violentmonkey

    Chrome：Tampermonkey 或 Violentmonkey

    Firefox：Greasemonkey、Tampermonkey 或 Violentmonkey

    Safari：Tampermonkey 或 Userscripts

    Opera：Tampermonkey 或 Violentmonkey

插件安装完成后，再安装脚本
